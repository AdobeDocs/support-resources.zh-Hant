# Workfront Admin Console登陸頁面

準備將您的組織加入Adobe Admin Console [移轉檢查清單](https://experienceleague.adobe.com/docs/workfront/using/administration-and-setup/admin-in-admin-console/prep-for-admin-console.html)：

**移轉檢查清單**

* 識別您要在其中新增Workfront的所需Adobe Admin Console。

  如果您已有其他Adobe產品，請確認您願意將Adobe Workfront新增至目前的Adobe Admin Console。 如果您需要更多資訊，請使用[此連結](https://helpx.adobe.com/tw/enterprise/using/admin-console.html)瞭解管理Adobe產品的最佳實務。

* 請參閱[下列連結](https://helpx.adobe.com/enterprise/using/deployment-planning.html)以取得部署的說明。
* 如果使用單一登入，您將需要將新的Adobe Admin Console連線至現有的SSO提供者。
* 確保移除重複的電子郵件，並僅使用有效的電子郵件地址。
* 如果您使用自訂API整合，請確認這些使用者擁有可以存取的有效電子郵件地址。

  Workfront建議將所有自訂整合更新為[利用OAuth2](https://experienceleague.adobe.com/docs/workfront/using/administration-and-setup/configure-integrations/create-oauth-application.html)。

設定好主控台後，請參閱此連結以檢視[如何在您的Adobe Admin Console中管理使用者](https://experienceleague.adobe.com/docs/workfront/using/administration-and-setup/add-users/create-manage-users/admin-console.html)。
