# Adobe Workfront支援概述

Adobe Workfront即將加入Adobe Admin Console Platform。

在接下來的幾個月中，我們將聯絡客戶，協助他們移轉至Adobe Identity和Admin Console，這是一個更安全、功能更強大的系統管理主控台。 以下是即將進行變更的一些優點：

* 使用單一Adobe身分驗證Workfront和其他產品。
* 在生產環境和沙箱或預覽環境之間單獨管理使用者權益。
* 為Workfront使用者利用多個SSO設定。
* 以管理其他Workfront產品的相同方式管理Adobe權益。

如需深入檢閱Adobe Identity和Admin Console，您可以參考此[支援頁面](https://helpx.adobe.com/tw/enterprise/admin-guide.html)。 [這裡](https://one.workfront.com/s/document-item?bundleId=the-new-workfront-experience&topicId=Content%2FAdministration_and_Setup%2FGet_started-WF_administration%2Factions-in-admin-console.htm&_LANG=enus)還有目前Workfront主控台與新Adobe主控台的比較。

## 客戶移轉細節

我們將與現有客戶聯絡，以排程從2022年7月開始進行移轉。  我們的移轉團隊支援同事將引導客戶完成整個程式、提供主控台設定建議，並提供必要的檔案連結，讓移轉過程儘可能簡單而輕鬆。

自2022年7月起的新客戶淨額會自動加入Adobe Admin Console，無需進一步動作。
