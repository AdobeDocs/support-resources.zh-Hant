# Adobe Admin Console中的協助工具

適用於企業和團隊。

瞭解Adobe Admin Console中的協助工具支援，該支援提供管理整個組織中Adobe許可權的集中位置。

## 作業系統協助工具支援

許多廣泛使用的作業系統的協助工具功能可用於Admin Console。 這些協助工具功能包括高對比和其他顯示設定。 不過，由於Admin Console目前的使用者介面，可能會有一些限制。

## 鍵盤支援

Admin Console上的大部分互動式內容都可單獨使用鍵盤來導覽和操作。 例如，您可以使用&#x200B;**[!UICONTROL Tab]**&#x200B;或&#x200B;**[!UICONTROL Shift]** + **[!UICONTROL Tab]**&#x200B;進行導覽與控制。 導覽至正確的連結或按鈕後，按下&#x200B;**[!UICONTROL Enter]**&#x200B;或&#x200B;**[!UICONTROL Return]**&#x200B;鍵以選取它。 仍可能會有一些例外，如[協助工具一致性報告](https://www.adobe.com/accessibility/compliance.html)中所述（_僅英文_）。

## 輔助技術支援

Admin Console對輔助技術（包括熒幕助讀程式和熒幕放大鏡）提供部分支援。 如需詳細資訊，請參閱[協助工具符合性報告](https://www.adobe.com/accessibility/compliance.html)。

## Admin Console支援哪些協助工具功能？

[Adobe Admin Console](https://adminconsole.adobe.com/)提供管理整個組織中Adobe許可權的集中位置。 它提供輔助技術支援、鍵盤支援，以及作業系統協助工具支援（少數例外）。

## 有回饋意見嗎？

Admin Console團隊持續致力於改善其協助工具和使用者體驗。 您可以在[企業與Teams社群](https://community.adobe.com/enterprise-teams-622)分享您對協助工具功能的意見反應。
