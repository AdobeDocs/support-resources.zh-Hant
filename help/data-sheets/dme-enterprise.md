# 企業支援Creative Cloud和Adobe Document Cloud

![圖示](assets/EnterpriseBanner.png)

Adobe 提供全方位的技術資源來支援您的業務，這些資源包含在您的 Adobe 企業訂閱中。ENTERPRISE支援計畫增強了此功能。 企業客戶將受益於我們的「指定支援工程師」服務，在此服務中，Adobe支援團隊中的指定技術聯繫人將與您和您的技術團隊合作，確保及時解決所有支援請求。

您的支援團隊還可以幫助協調和安排交付額外的企業優勢，確保在最關鍵的時刻將業務中斷降至最低。 企業支援計畫客戶還可以利用我們詳細而深入的技術產品文檔和當前發行說明。
