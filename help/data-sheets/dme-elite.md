# 精英支援Creative Cloud和Adobe Document Cloud

![圖示](assets/Elitebanner.png)

Adobe 提供全方位的技術資源來支援您的業務，這些資源包含在您的 Adobe 企業訂閱中。ELITE支援計畫增強了這一能力。 ELITE客戶可以訪問指定支援工程師和技術客戶經理，他們與您合作提供同類最佳的主動和被動支援，同時在Adobe支援團隊中充當您指定的技術聯繫人。

憑借在您適用的Creative Cloud和Document Cloud解決方案方面的豐富經驗，您的支援團隊將致力於確保無論您的支援需求多麼複雜，Adobe支援團隊都將與您一起，確保您在適用的Adobe解決方案方面的投資實現最大價值，並幫助您在問題發生之前避免問題。 您還可以利用我們詳細而深入的技術產品文檔和當前發行說明。
