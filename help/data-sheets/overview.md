# Adobe支援計畫概述

Adobe支援計畫不僅包括被動式問題解決，還能為您提供新類型的個人化體驗，讓您的業務步驟領先於任何問題。 與我們的深層技術專業知識和創新的支援模式合作，將提高您的技術效能，並幫助您實現Adobe解決方案的潛力。
